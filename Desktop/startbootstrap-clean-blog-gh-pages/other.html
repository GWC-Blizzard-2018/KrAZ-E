About Us

Girls Not Brides

Girls Not Brides Awareness is dedicated to bringing awareness to the Girls Not Brides movement. Please help us support our cause and help out girls in India by donating.

Girls Who Code

Girls Who Code is an organization created to empower girls everywhere by teaching them how to code and closing the gender gap in technology. A group of girls from their program created this site.
For more information, visit the Girls Who Code Website
